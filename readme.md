# Yet another text editor

## Controls

| Key Combo        | Action              |
| ---------------- | ------------------- |
| CTRL + S         | Save                |
| CTRL + SHIFT + S | Save as             |
| CTRL + O         | Open                |
| CTRL + LEFT      | Move Left a word    |
| CTRL + RIGHT     | Move Right a word   |
| CTRL + Plus      | Zoom in             |
| CTRL + Minuse    | Zoom Out            |
| ALT + UP         | Move line up        |
| ALT + DOWN       | Move Line Down      |
| SHIFT + ARROW    | Select              |
| CTRL + C         | Copy Line/Selection |
| CTRL + X         | Cut Line/Selection  |
| CTRL + V         | Paste               |
